## 🧩 Үндсэн Боломжууд

### 🎮 Хэрэглэгчийн тал (тоглогчид)

**Заавал байх ёстой:**
- Тоглоомын газруудыг үзэх
- Дараах мэдээллийг харах:
  - VIP / өрөөнүүд
  - PC-ийн тодорхойлолт (specs)
  - Цагийн / сессийн үнэ
- Дараах зүйлс сонгох:
  - Цаг өрөө
  - Үргэлжлэх хугацаа (жнь: 3ц, 5ц)
- 👉 **"Захиалга илгээх"** товч дарах

**Захиалсны дараа — статус харах:**

| Статус | Тайлбар |
|---|---|
| ⏳ Хүлээгдэж байна | Газрын хариу хүлээж байна |
| ✅ Баталгаажсан | Газар зөвшөөрсөн |
| ❌ Татгалзсан | Газар татгалзсан |

---

### 🏢 Тоглоомын газрын тал

**Хяналтын самбар (Dashboard):**
- Захиалгын хүсэлтүүдийг харах
- ✅ Зөвшөөрөх / ❌ Татгалзах
- ✅ Дууссан гэж тэмдэглэх

---

### 🧠 Системийн хяналт

| Талбар | Утга |
|---|---|
| `booking_id` | Давтагдашгүй захиалгын дугаар |
| `center` | Тоглоомын газрын нэр |
| `time` | Захиалсан огноо, цаг |
| `estimated_price` | Урьдчилсан дүн |
| `status` | pending / confirmed / completed / rejected |

## ⚙️ Захиалгын Урсгал

```
Хэрэглэгч → Газар сонгох → Өрөө сонгох → Цаг сонгох
    ↓

    ↓

    ↓
✅ Дууссан гэж тэмдэглэнэ
```

## 🔒 Урвуулалт Хаах Систем

Төлбөргүй учир дараах дүрмүүд **заавал** байна:

- 🚫 Нэг хэрэглэгч нэг идэвхтэй захиалгатай байж болно
- 📋 Ирэхгүй байдлыг хянана (no-show tracking)
- ⏰ 15 минут хоцорсон бол **автоматаар цуцлагдана**

> ⚠️ **3 удаа ирэхгүй байвал = Түр хориглол**

## ✨ Шинэлэг Нэмэлт Боломжууд

### 🗺️ Интерактив Газрын Зураг
- Бүх газруудыг Google Maps / Mapbox дээр харуулна
- Хэрэглэгч байршлаасаа ойр газруудыг олох боломж
- Өрөө байгаа / байхгүй байдлыг шууд харуулна:
  - 🟢 Хоосон өрөө байна
  - 🔴 Бүгд дүүрсэн

### ⚖️ PC Газруудыг Харьцуулах
- 2–3 газрыг зэрэг харьцуулах хэрэгсэл
- Харьцуулах зүйлс:
  - 💻 PC specs (GPU, RAM, Монитор)
  - 💰 Цагийн үнэ
  - ⭐ Дундаж үнэлгээ (rating)
  - 🕐 Хоосон слот
  - 📍 Зай (байршлаас)

### ⭐ Үнэлгээ & Сэтгэгдэл
- Захиалга дуусмагц хэрэглэгч үнэлгээ өгнө (1–5 ⭐)
- Газрын хуудсанд дундаж үнэлгээ харагдана
- Газруудыг дундаж үнэлгээгээр эрэмбэлэх боломж

### 📊 Хэрэглэгчийн Статистик
- Нийт тоглосон цаг
- Хамгийн их зочилдог газар
- Дуртай газрыг хадгалах (❤️ favorites)

### 🔔 Ухаалаг Мэдэгдэл
- Захиалга 30 минутын өмнө сануулга
- Хоосон слот гарвал мэдэгдэл (waitlist)
- Дуртай газар урамшуулал зарлавал мэдэгдэл

### 📅 Урьдчилан Захиалах
- 7 хоног хүртэл урьдчилан захиалах боломж
- Долоо хоног бүрийн тогтмол захиалга (жнь: Бямба гаригт 3ц)
- **"Дахин захиалах"** — өмнөх захиалгаа хурдан давтах
